# Atlas Mission Control Dashboard

A Linear-inspired Next.js dashboard that surfaces Paul & Atlas' day-to-day context: daily memory logs, agent pipelines, and quick actions to keep the four-agent squad in sync.

## Getting Started

```bash
cd mission-control-dashboard
npm install
npm run dev
```

Then open http://localhost:3000 to view the dashboard.

The dashboard automatically looks for daily memory markdown files in `../memory` (relative to this project root). Populate that directory to see the "Latest Logs" section update in real time.
