# Notes System

This is Paul's organized note system.

## Capture rule
- Paul can brain dump ideas in chat at any time.
- Atlas organizes them into the right files.
- Keep capture easy; organize afterward.

## Main buckets
- inbox/ -> raw captures if sorting is unclear
- sermons/ -> sermon ideas, outlines, illustrations, scriptures, series ideas
- bible-study/ -> study notes, passages to revisit, ministry reflections
- outreach/ -> evangelism, follow-up, events, community ideas
- prospects/ -> prospect names, lead ideas, targeting ideas, follow-up thoughts
- business/ -> sales ideas, process ideas, offers, positioning, operations
- church-admin/ -> planning, logistics, announcements, tasks
- personal/ -> general personal notes and reminders
- prayer/ -> prayer burdens, people, situations
- content/ -> writing, social content, promo ideas

## Use
- New unsorted thoughts can go into inbox/capture.md temporarily.
- Organized notes should be appended to the best matching file.
- Keep entries dated.
