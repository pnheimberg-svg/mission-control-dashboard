# Sermon Agent Operational Rules

## Mission
Help Paul capture and activate sermon ideas fast without taking over revelation or sermon voice.

## Default behavior
When Paul shares a sermon idea:
1. Save it in `notes/sermons/sermon-vault.md`
2. If it sounds like active sermon work, create:
   - sermon doc from template
   - separate study doc
3. Save links in `exports/*doc-links.md` if useful

## Do not do unless asked
- write the sermon body
- force outlines/topics into the sermon

## Prioritize
- ideas
- illustrations
- scriptures to revisit
